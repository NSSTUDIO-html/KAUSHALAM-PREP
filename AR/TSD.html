Time, Speed & Distance

Concepts • Formulas • Shortcuts • Practice

Basic Formula

Speed = Distance / Time
Distance = Speed × Time
Time = Distance / Speed

Important Concepts

Relative Speed

Same direction → subtract speeds
Opposite direction → add speeds

Trains

Distance includes train length

Boats & Streams

Downstream → speed increases
Upstream → speed decreases

Average Speed

Avg Speed = Total Distance / Total Time

Shortcut Tricks

✔ Convert km/hr → m/s: multiply by 5/18

✔ Convert m/s → km/hr: multiply by 18/5

✔ Same distance → Avg speed = (2xy)/(x+y)

Example: speeds x and y

Example

Speed = 60 km/hr, Time = 2 hr

Distance = 60 × 2 = 120 km

Answer = 120 km

Ratio & Proportion

Concepts • Formulas • Shortcuts • Practice

Ratio

Ratio compares two quantities.

a : b = a / b

Example: 2 : 3 means 2/3

Proportion

Equality of two ratios.

a : b = c : d

Example: 2:3 = 4:6

Important Formula

If a : b = c : d → a × d = b × c

Types

Direct Proportion

Increase → Increase

Example: More workers → more work

Inverse Proportion

Increase → Decrease

Example: More workers → less time

Shortcut Tricks

✔ Multiply or divide both terms → ratio remains same

✔ Convert ratio into fraction for easy solving

✔ Sum of ratios → total parts

Example: 2:3 → total parts = 5

Examples

If ratio is 2:3 and total = 100

Total parts = 5

First = 40, Second = 60

Practice Questions

1. Divide 60 in ratio 2:3

2. If a:b = 3:4 and b:c = 2:5, find a:c

3. Solve: 2:5 = x:20

4. A:B = 4:7, total = 110 → find A

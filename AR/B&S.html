Boats & Streams

Concepts • Formulas • Shortcuts • Practice

Basic Concept

Downstream → Speed increases
Upstream → Speed decreases

Important Formulas

✔ Downstream Speed = Boat Speed + Stream Speed

✔ Upstream Speed = Boat Speed - Stream Speed

✔ Boat Speed = (Down + Up) / 2

✔ Stream Speed = (Down - Up) / 2

Shortcut Tricks

✔ Use average method for fast solving

✔ Same distance → use ratio method

✔ Convert units if needed (km/hr ↔ m/s)

Example

Downstream = 10 km/hr, Upstream = 6 km/hr

Boat speed = (10 + 6)/2 = 8 km/hr

Stream speed = (10 - 6)/2 = 2 km/hr

Practice Questions

1. Find boat speed if downstream = 12, upstream = 8

2. Stream speed if down = 15, up = 9

3. Find time for upstream travel

4. Same distance → find avg speed

Number System (Aptitude)

Concepts + Shortcuts + Exam Tricks

Types

  • Decimal (Base 10)
  • Binary (Base 2)
  • Octal (Base 8)
  • Hexadecimal (Base 16)

Core Formula

N = dₙbⁿ + dₙ₋₁bⁿ⁻¹ + ... + d₀b⁰

Important Topics for Exams

Divisibility Rules

  • 2 → Last digit even
  • 3 → Sum of digits divisible by 3
  • 5 → Ends with 0 or 5
  • 9 → Sum of digits divisible by 9

Remainder Theorem

(a + b) mod n = [(a mod n) + (b mod n)] mod n

Unit Digit Trick

Focus only on last digit pattern cycles.

Example: 2ⁿ → cycle (2,4,8,6)

HCF & LCM

Product of numbers = HCF × LCM

Shortcut Tricks

✔ To find remainder → Use modulo shortcut

✔ Large powers → Use cyclicity

✔ Binary conversion → Divide by 2

✔ Fast square → Use (a+b)² formula

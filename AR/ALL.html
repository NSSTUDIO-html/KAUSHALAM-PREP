Basic Concept

Important Identities

✔ (a + b)² = a² + 2ab + b²

✔ (a - b)² = a² - 2ab + b²

✔ a² - b² = (a - b)(a + b)

✔ (a + b)³ = a³ + b³ + 3ab(a + b)

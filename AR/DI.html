📈 Data Interpretation - Complete Concepts

Bar Graph

Represents data using rectangular bars (vertical/horizontal).

Tip: Compare heights to find values quickly.

Pie Chart

Shows percentage distribution in circular form.

Formula: Value = (Angle / 360) × Total

Line Graph

Represents trends over time using lines.

Tip: Focus on increase/decrease patterns.

Table DI

Data presented in rows and columns.

Tip: Scan required data only (don’t read everything).

Caselet DI

Data given in paragraph form instead of table.

Tip: Convert into table for easier solving.

Mixed Graphs

Combination of multiple charts (bar + line, pie + table).

Tip: Analyze one graph at a time.

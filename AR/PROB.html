Probability

Concepts • Formulas • Shortcuts • Practice

Basic Concept

Probability = Favorable Outcomes / Total Outcomes

Important Formulas

✔ P(E) = n(E) / n(S)

✔ 0 ≤ Probability ≤ 1

✔ P(Not E) = 1 - P(E)

Types of Events

Independent Events

One event does not affect another

Dependent Events

One event affects another

Shortcut Tricks

✔ Use complement method → faster solving

Example: P(not getting head)

✔ For multiple events → multiply probabilities

✔ Total probability always ≤ 1

Example

Find probability of getting head in coin toss

Favorable = 1, Total = 2

P = 1/2

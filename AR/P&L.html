Profit & Loss

Concepts • Formulas • Shortcuts • Practice

Cost Price (CP)

Price at which an item is bought

Selling Price (SP)

Price at which an item is sold

Profit

Profit = SP - CP

Loss

Loss = CP - SP

Important Formulas

✔ Profit% = (Profit / CP) × 100

✔ Loss% = (Loss / CP) × 100

✔ SP = CP × (100 + Profit%) / 100

✔ SP = CP × (100 - Loss%) / 100

Discount

✔ Marked Price (MP) = Original price

✔ Discount = MP - SP

✔ Discount% = (Discount / MP) × 100

Shortcut Tricks

✔ Profit% and Loss% always calculated on CP

✔ Successive profit/loss → multiply factors

Example: +20% then +10% → 1.2 × 1.1

✔ Same % gain & loss → always loss

Examples

CP = 100, SP = 120

Profit = 20

Profit% = 20%

Practice Questions

1. CP = 200, SP = 250 → find profit%

2. Find SP if CP = 100 and profit = 20%

3. MP = 500, discount = 10% → find SP

4. Profit = 50, CP = 250 → find profit%

Average

Concepts • Formulas • Shortcuts • Practice

Basic Concept

Average = Total Sum / Number of Values

Important Formulas

✔ Average = Sum / Count

✔ Sum = Average × Count

✔ New Avg = (Old Sum ± Change) / New Count

Types of Questions

Simple Average

Find average of numbers directly

Change in Average

Adding/removing values changes average

Shortcut Tricks

✔ If numbers are equally spaced → Average = Middle value

Example: 10,20,30,40,50 → Avg = 30

✔ Adding a number greater than avg → avg increases

✔ Adding a number smaller than avg → avg decreases

✔ Replace method → saves time in exams

Examples

Average of 5 numbers = 20

Total sum = 5 × 20 = 100

If one number replaced by 30 → new sum = 110

New avg = 110 / 5 = 22

Practice Questions

1. Find avg of 10, 20, 30, 40

2. Avg of 5 numbers is 15 → find sum

3. Avg increases by 2 when one number added → find number

4. Replace 20 with 40 → find new avg
